This is an example of an uncommon HTML bug related to the use of the <object> tag.

The <object> tag is used to embed external data into an HTML page. However, if the data source is not available or there is an error loading the data, the <object> tag might not render correctly.

<object data="invalid-source.pdf" type="application/pdf"><embed src="invalid-source.pdf" type="application/pdf" /></object>

In this example, the data and src attributes point to an invalid source which might lead to a blank space or a broken icon instead of the expected PDF document.
